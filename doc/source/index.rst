==========
PyAEDT-MCP
==========

.. toctree::
   :hidden:
   :maxdepth: 3

   getting_started/index
   user_guide/index
   api/index
   examples/index
   changelog

PyAEDT-MCP is an MCP server for Ansys Electronics Desktop (AEDT). It gives an
MCP client a focused set of AEDT tools, a persistent PyAEDT-backed Python
session, and a simple startup flow for launching or attaching to AEDT over
gRPC.
