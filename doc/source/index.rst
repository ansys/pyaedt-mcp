PyAEDT-MCP is an MCP (Model Context Protocol) server for Ansys Electronics Desktop (AEDT), built using `FastMCP <https://github.com/PrefectHQ/fastmcp>`_. It provides a focused set of tools and a persistent PyAEDT-backed Python session.
When the MCP starts, it can optionally automatically connect to a running AEDT instance.
